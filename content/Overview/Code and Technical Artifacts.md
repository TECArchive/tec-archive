# Code & Technical Artifacts

TEC produced a large body of software through smart contracts, governance and economic apps, dashboards, bots, data tooling, research code, and brand assets. The archive preserves **63 source repositories** mirrored from TEC's development organizations. This page catalogs them by theme.

> These repositories are preserved as historical snapshots. They are not maintained and may depend on deprecated services. Code is included as part of the record, not as a working deployment.

## Token economy & the Commons configuration

- `augmented-bonding-curve-model` — model of the Augmented Bonding Curve
- `commons-config-backend`, `commons-config-frontend`, `commons-config-dashboard`, `commons-config-website`, `commons-config-proposals` — the Commons Configuration Dashboard and supporting services used to design and parameterize the upgrade
- `tec-abc-dashboard`, `tec-abc-subgraph` — Augmented Bonding Curve dashboard and subgraph
- `FundraisingModel`, `proposal-inverter`, `revenue-sharing`, `tec-swap` — economic models and mechanisms
- `tec-dao`, `tec-template`, `token-manager-app`, `marketplace-app` — DAO and Aragon-style app infrastructure

## Hatch & governance configuration

- `TEC-Hatch-Vote`, `Test-TEC-Hatch-Vote` — the Hatch voting app and its test version
- `Token-Log-Hatch-Params`, `TokenLog-SoftGov`, `tqf` — parameter logging, soft governance, and token quadratic funding tooling

## Praise, rewards & SourceCred

- `praise_backend`, `praise-quantification`, `praiseanalysis`, `praise_RewardAnalysis`, `praise_DiscordChannelRevision`, `praise_google_spreadsheets_export` — the Praise contribution-recognition system and its analytics

- `tec-sourcecred`, `tec-sourcecred-dashboard`, `pollen-cred-fetcher` — SourceCred-based contribution measurement

## Conflict transformation
